# Caso de uso: Gestión de inscripciones

Este módulo agrupa la documentación funcional relacionada con el registro de estudiantes en el sistema SEP MUSES. La información se separa en dos casos de uso complementarios:

- **CU-001 Inscripción manual**: Permite registrar estudiantes uno a uno mediante un formulario.
- **CU-002 Inscripción masiva desde CSV**: Automatiza el registro a partir de un archivo CSV, muestra progreso en tiempo real, persiste el lote procesado en el navegador y ofrece una vista de detalle por registro.

Cada documento detalla actores, precondiciones, flujos y reglas específicas con el fin de apoyar el diseño y la implementación de las interfaces correspondientes.
