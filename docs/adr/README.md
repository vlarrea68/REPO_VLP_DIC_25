# Registros de Decisiones de Arquitectura (ADRs)

Este directorio contiene los Registros de Decisiones de Arquitectura (ADRs) para el proyecto SEP-MUSES. Un ADR es un documento breve que captura una decisión arquitectónica importante junto con su contexto y consecuencias.

El propósito de los ADRs es crear un registro histórico del "porqué" detrás de las decisiones clave, ayudando a los miembros actuales y futuros del equipo a entender la evolución de la arquitectura del sistema.
