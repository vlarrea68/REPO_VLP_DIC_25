# Cronograma de Ejecución de Pruebas y Generación de Reportes

## 1. Propósito

Este documento detalla el cronograma para la ejecución de cada ciclo de pruebas y la generación de sus respectivos reportes de evidencia. Sirve como una guía para que el equipo de QA y los líderes de proyecto planifiquen y asignen recursos, con fechas de inicio y finalización claras para cada actividad.
